Tämä on yksinkertainen mutta tehokas selainsovellus, joka on rakennettu Electronilla ja Typescriptillä.

### Vaatimukset

- Node.js 16+ 
- npm tai yarn

### Asennus

```bash
npm install
```

### Kehitysympäristön käynnistäminen

```bash
npm run dev
```

### Rakentaminen

```bash
npm run build
```

## Projektin rakenne

```
src/
├── components/      - React-komponentit
├── hooks/          - React-hookit
├── lib/            - Apufunktiot ja Business Logic
├── utils/          - Hyödylliset utilityt
└── styles/         - Tyylit

electron/           - Electron main-prosessi
```
